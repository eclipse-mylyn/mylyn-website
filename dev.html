Mylar Developer Documentation

If you are interested in extending Mylar please email mylar-dev@eclipse.org.  Mylar is in a pre-release stage, evolving rapidly, and provides no fixed APIs.  We expect to announce some stability in the APIs before the end of summer, and until then the policy is extend at your own risk. 

building

Mylar requires Eclipse SDK 3.1 and JDK 1.5.  To set up a workspace for mylar development:
- Add CVS location: dev.eclipse.org /home/technology
- Check out all of the projects under org.eclipse.mylar

monitor

The monitor component is used for running the user study.  For the duration of the study it will be hosted at UBC. 
- Add CVS location: cascade.cs.ubc.ca /isd/se2/project/mylar/cvs
- Check out the 4 org.eclips.mylar.monitor projects

conventions

For handling exceptions and logging use MylarPlugin.log(..) or MylarPlugin.fail(..).
