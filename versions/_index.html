Mylyn Versions

The scope of the Mylyn Versions project is Eclipse integration for source code and configuration management tools and bi-directional linking to change management tools. This project and APIs it provides builds on the existing Eclipse "Team" APIs.

Incubation

Mylyn Versions is in the Incubation Phase.
