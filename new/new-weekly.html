New & Noteworthy for Mylyn Weekly Builds

Also see the New & Noteworthy for: Mylyn 3.1

  • The presentation of the notification popup has been improved (bug 274656)
  • The JIRA task editor now supports logging of work (bug 262799)
  • The search view now supports sorting (bug 216150)
  • The submit button of the Bugzilla Task Editor is now always visible (bug 266600)
  • The working set switcher now affects all windows (bug 250497)
  • Stack traces are now displayed in a pre-formatted block when WikiText is enabled (bug 272757)
  • Connectors can be easily installed from the Add Task Repository wizard (bug 272621)
  • Orphaned elements are now displayed on the context tab (bug 174413)
  • Task List tooltips element now use a gradient (bug 276581)
  • Improved screen capture UI (bug 266123)
  • The submit button is now displayed in the header of the Bugzilla editor (bug 274790) Note: Changes made for bug 266600 that moved the submit button into a always visible footer part have been reverted.
  • The task editor header has a big editable priority control now (bug 272093)
  • The local task editor has been improved and is now consistent with repository task editors (bug 211096)
  • The integrated bug reporting has been improved, see Help - Report Bug or Enhancement (bug 212209)
  • The attachment dialog now supports spell checking and wiki markup (bug 245187)
  • The open and activate task dialogs have an I'm feeling lazy button (bug 248490)
