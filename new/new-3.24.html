Supported Repository Versions

Connectors included in Mylyn 3.24 support the following repository versions:
  • Bugzilla 4.0 - 4.4.13, 5.0.4
  • Gerrit 2.9.4 - 2.11.3
  • Hudson 3.0 - 3.3.3
  • Jenkins 1.509 - 2.32.3
