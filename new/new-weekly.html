New in Mylyn 3.2

Connector Discovery

Connector Install Wizard

From the Add Task Repository dialog you can access the Connector Discovery wizard for simple installation of Mylyn extensions. To lauch the Connector Discovery wizard, open the Task Repositories view and click the New Repository button. In the resulting dialog, click the "Install More Connectors..." button to launch Connector Discovery. For information on how connectors are made available via the Connector Discovery mechanism stay tuned to the Mylyn Integrators mailing list.

Integrated Bug Reporting

Specify Support Providers

If you choose to report an error from the error log (right click > Report as Bug), the appropraite repository will be chosen to file the bug against for you. When using Help > Report Bug or Enhancement..., you are presented with a wizard to help pick the correct repository based on installed features. Integrators can make use of an extension point (org.eclipse.mylyn.tasks.bugs.support) to specifying support providers and supported products.

Task-Focused Interface

Hidden Context Members

The context page of the Task Editor shows elements that can not be presented in the common navigator in a separate table.

Task List

Improved Notification and Tooltip UI

Change notifications and tooltips to use a gradient for the background to improve their visual appearance.

Toggle Auto Synchronization per Query

Toggle whether a query is included in automatic background query synchronizations. Right click on the desired query and deselect "Synchronize Automatically" to ensure this repository does not get synchronized in the background.

Improved Sorting and Grouping Capability

Sort both the Task List and task search results view by multiple sort levels. The Sort dialog allows the user to choose how the task list folders are grouped:

  1. none - alphabetical
  2. Category and Query - categories grouped separately from queries, then sorted alphabetically
  3. Category and Repository - Unmatched/Unsubmitted categories grouped with queries of same repository, queries then sorted alphabetically

Task Editor

Unified Local and Repository Editor

The appearance of the editor for local and repository tasks has been streamlined and unified. Connector authors can now enable repositioning of the personal planning controls and notes to be folded into a new Private section within the repository task editor page itself.

Screen Capture Improvements

The screen capture wizard supports improved basic image editing and has capabilities for importing images from files and the clipboard.

Local Task and Attachment Spell Checking/WikiText

The local task editor and attachment dialog support both spell checking and wikitext markup. In a local Task editor, click the 'Local' label in the editor header to configure the wiki syntax.

WikiText

Pre-formatted Stack Traces

To improve consistency of layout, stack traces are now displayed in a pre-formatted block.

WikiText to PDF

The WikiText component now supports export to XSL Formatting Objects (XSL-FO) and PDF in addition to the other formats curretly supported (HTML, Eclipse Help, DocBook, DITA)

JIRA Connector

Work Log

The Task Editor supports adding work log entries.

Trac Connector

Trac 0.11

The Trac Connector now supports Trac 0.11 if the latest version of Trac's XmlRpcPlugin is installed on the repository.

Select Owner from Drop-down

The Task Editor shows a drop-down for re-assigning tickets if restrict_owner is enabled for a repository.
